A lightweight and customizable WooCommerce side cart plugin with Ajax functionality. Display cart items in a modern side panel with controls to add, remove, and manage products without reloading the page.

== Description ==

**Kartly SideCart for WooCommerce** is a sleek, Ajax-enabled cart sidebar that provides a smooth shopping experience. As soon as a user adds a product to their cart, a stylish cart slides into view, showing all items in real-time.

**Features:**

- Display WooCommerce cart contents in a responsive side panel
- Ajax add-to-cart and remove-from-cart functionality
- Update product quantities without page reload
- Buttons for: Continue Shopping, View Cart, and Checkout
- Lightweight and customizable via CSS
- Works with most WooCommerce themes

**Great for improving UX** by reducing cart page reloads and encouraging more conversions.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory or install via the WordPress plugin installer.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. The side cart will automatically appear on the frontend.
4. Optionally, configure design settings in **Appearance > Customize > KM SideCart** or your plugin settings page.

== Frequently Asked Questions ==

= Does this work with all themes? =  
The plugin is designed to work with most WooCommerce-compatible themes. Minor CSS adjustments may be needed for full compatibility.

= Can I customize the appearance? =  
Yes, you can override styles using custom CSS, or adjust settings from the plugin settings panel.

= Does it support variable products? =  
Yes. All product types added to the WooCommerce cart are supported.
